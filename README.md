# MyWorkbench

Starter template based on [Vite](https://vitejs.dev/guide/) with a small boilerplate page incl. **routing & theming**.

**Includes**:

- React 18
- React Router
- React Query (comming soon)
- Axios
- Material-UI
- Recharts
- Iconify
